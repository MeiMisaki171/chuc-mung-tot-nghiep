# Chúc mừng tốt nghiệp 🎓

Trang web chúc mừng tốt nghiệp được chuyển từ HTML sang React + Vite.

## Chỉnh sửa nhanh

Mở `src/config.js` để đổi tên người nhận:

```js
export const CONFIG = {
  tenBan: 'Bạn',
  nganhHoc: 'Đại học',
  tenNguoiGui: 'Những người luôn tự hào về cậu',
}
```

## Chạy local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
